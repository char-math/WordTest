# 词汇量测评工具 - 项目理解文档

## 一、项目概述

### 1.1 项目简介

本项目是一个基于 Node.js + Express + SQLite 的在线词汇量测评系统。用户可以通过回答单词释义选择题来评估自己的词汇水平，系统支持小学、初中、高中三个难度级别，并提供 AI 对战和用户匹配 PK 功能，增加学习的趣味性。

### 1.2 项目目标

- 提供一种有趣、高效的词汇量评估方式
- 通过 PK 对战增加学习动力和趣味性
- 记录用户学习轨迹，帮助了解自身进步
- 支持错题本功能，巩固薄弱知识点

### 1.3 目标用户

- 在校学生（小学、初中、高中）
- 希望扩充英语词汇量的学习者
- 需要系统评估自身词汇水平的用户

## 三、核心功能模块

### 3.1 用户认证系统

实现完整的用户注册、登录、登出功能：

- **用户注册**：支持用户名和密码注册，密码使用 bcrypt 加密存储
- **用户登录**：验证凭证后返回 JWT Token
- **Token 管理**：前端存储在 localStorage，请求时自动携带
- **自动登录**：页面刷新时自动恢复登录状态
- **权限区分**：普通用户和管理员角色

### 3.2 词汇量测评

核心测评功能，包含以下特点：

- **三级词库**：小学(505词)、初中(1110词)、高中(1866词)
- **智能出题**：根据用户当前水平动态调整题目难度
- **多选题形式**：每题6个选项(1正确+4干扰+不认识)
- **实时统计**：正确数、错误数、正确率
- **结束条件**：完成50题 / 连续答错4题 / 累计答错7题
- **词汇量评估**：基于正确数计算估算词汇量

### 3.3 AI 对战模式

与 AI 对手实时对战：

- **三档难度**：简单、中等、困难
- **AI 模拟**：根据难度呈现不同答题速度和正确率
- **抢答机制**：先答对者得分
- **限时答题**：每题30秒倒计时
- **实时比分显示**

### 3.4 用户匹配 PK

真人对战功能：

- **房间匹配**：用户可创建或加入房间
- **实时同步**：题目和时间同步显示
- **抢答得分机制**
- **胜负判定**：答题数和正确率综合评估

### 3.5 错题本系统

错题收集与复习：

- **自动收集**：测评过程中答错的单词自动加入错题本
- **错题复习**：支持选词填空模式复习
- **移除机制**：复习正确后自动从错题本移除
- **本地存储**：使用 localStorage 持久化

### 3.6 个人中心

用户功能聚合：

- **个人信息**：用户名、角色、等级
- **测评统计**：测评次数、最高词汇量、平均正确率
- **历史记录**：查看历次测评结果
- **水平变化**：词汇量变化趋势图表
- **错题本管理**：查看和复习错题

### 3.7 管理员后台

系统管理功能：

- **用户管理**：查看用户列表、禁用/启用账号
- **词库管理**：增删改查单词数据
- **测试记录**：查看所有用户的测评记录
- **数据统计**：系统使用数据统计

## 六、安全设计

### 6.1 密码安全

- 使用 bcrypt 算法加密存储密码
- 加盐处理，防止彩虹表攻击
- 登录失败次数限制

### 6.2 身份认证

- JWT Token 过期时间：7天
- Token 存储在 localStorage
- 每次请求自动携带 Authorization 头
- Token 过期自动跳转登录页

### 6.3 权限控制

- 管理员接口需要 admin 角色
- 用户接口需要有效 Token
- 防止越权访问其他用户数据

## 九、常见问题

### Q1: 如何重置管理员密码？
A: 可以通过 Node.js 脚本直接修改数据库中的密码字段，使用 bcrypt 重新加密新密码。

### Q2: 如何增加新的单词到词库？
A: 登录管理员后台，在"词库管理"页面添加新单词，或通过 API 接口批量导入。

### Q3: 支持哪些浏览器？
A: 支持 Chrome、Firefox、Edge、Safari 的最新两个版本。

### Q4: 数据存在哪里？
A: 所有数据存储在 SQLite 数据库文件 database.sqlite 中，可直接备份该文件。
